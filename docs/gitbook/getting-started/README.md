# Getting Started

Welcome to OpenACP! OpenACP lets you chat with AI coding agents — like Claude Code, Gemini CLI, or Copilot — directly from Telegram, Discord, or Slack. No browser tab switching, no terminal babysitting.

Pick where to start:

- **[What is OpenACP?](what-is-openacp.md)** — New here? Start with the big picture.
- **[Quick Start](quick-start.md)** — Install OpenACP, set it up, and chat with an AI agent in ~10 minutes.
- **[For Contributors](for-contributors.md)** — You want to hack on OpenACP, add a feature, or fix a bug.
