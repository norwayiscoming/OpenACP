# Troubleshooting

Having trouble? Find solutions to common issues below.

**Start here: run `openacp doctor`**

The built-in diagnostic command checks your configuration, bot tokens, agent commands, and platform connectivity automatically. It catches the majority of setup issues in seconds:

```bash
openacp doctor
```

Each check reports a pass, warning, or failure with a specific message — fix the reported issues first before diving into the guides below.
