# Using OpenACP

Learn how to interact with AI agents through your chat platform.

OpenACP connects AI coding agents to Telegram, Discord, and other messaging platforms. Once installed and configured, you control everything — creating sessions, managing agents, approving permissions, and more — directly from chat.

- [Chat Commands](chat-commands.md) — Full command reference for Telegram and Discord
- [Sessions](sessions.md) — What sessions are, how to create and manage them
- [Agents](agents.md) — Browse, install, and switch between AI agents
- [Permissions](permissions.md) — How agent permission requests work
- [Voice and Speech](voice-and-speech.md) — Send voice messages and receive spoken responses
- [Files and Media](files-and-media.md) — Send files and images to agents
